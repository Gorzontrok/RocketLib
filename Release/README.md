# RocketLib

This is a collection of useful tools for modding Broforce.

## Library Features

- Custom menu library with multiple layout options (vertical, horizontal, grid, paginated) and menu elements
- Custom trigger library for creating new trigger actions for Broforce's level editor system
- Extension methods for GameObject, Component, Unit, Projectile, Harmony, etc
- Collections for heroes, levels, pickups, and doodads
- Keybinding system with keyboard and controller support
- Debug drawing utilities for lines, rectangles, and crosshairs
- Tools for loading sprites and sounds

## User Features

- On-screen logger with color-coded messages and automatic log files (F7 to clear)
- Fix for the UMM mod manager window disappearing
- Fix for the bug where the cursor in the LevelEditor was stuck
